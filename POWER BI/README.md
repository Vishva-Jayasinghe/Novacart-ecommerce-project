# 📊 NovaCart Analytics Dashboard (Power BI)

A modern **E‑commerce Analytics Dashboard** built using **Microsoft
Power BI** to analyze sales performance, customer behavior, and
seller/shipping efficiency.\
This project demonstrates **data modeling, DAX measures, interactive
dashboards, and business insights** for an online retail platform.

------------------------------------------------------------------------

## 🚀 Project Overview

The **NovaCart Analytics Dashboard** provides insights into three major
business areas:

1.  **Sales Performance**
2.  **Customer Behavior**
3.  **Seller & Shipping Operations**

The dashboard uses **interactive slicers, KPIs, and visualizations** to
help stakeholders quickly understand business performance.
